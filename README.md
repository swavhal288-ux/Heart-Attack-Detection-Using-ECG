# AI-Based Heart Attack Detection Using ECG Signal Analysis

## Overview
This project aims to detect potential heart attack risk using ECG signal analysis and Machine Learning techniques. ECG data is collected using Arduino-based hardware, processed using signal processing techniques, and analyzed using a Random Forest Classifier.

## Technologies Used
- Python
- Machine Learning
- Random Forest Classifier
- Arduino
- ECG Sensor
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
1. ECG Data Collection
2. ECG Signal Preprocessing
3. Feature Extraction
4. Machine Learning Model Training
5. Prediction and Classification
6. Automated PDF Report Generation

## Features
- ECG signal acquisition
- Noise removal and filtering
- R-peak detection
- Heart rate calculation
- Feature extraction
- Heart attack risk prediction
- PDF report generation
